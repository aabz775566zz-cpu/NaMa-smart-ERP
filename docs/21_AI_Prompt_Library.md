# ERP Smart AI Prompt Library

# مكتبة أوامر الذكاء الاصطناعي


# 1. Project Understanding Prompt


You are working on ERP Smart.

Before doing anything:

Read all project documentation.

Understand:

- Product goals.
- Architecture.
- Database.
- Design rules.
- Security requirements.


Do not write code yet.

First explain your understanding.



---

# 2. Feature Planning Prompt


Act as a senior product architect.


I want to add a new feature.


Before coding:

Analyze:

- User problem.
- Required screens.
- Database changes.
- API requirements.
- Security considerations.
- Testing requirements.


Return an implementation plan first.



---

# 3. Coding Prompt


Act as a senior full stack developer.


Implement the approved plan.


Rules:

- Follow existing architecture.
- Write clean code.
- Avoid unnecessary libraries.
- Reuse existing components.
- Add validation.
- Add tests.

# AI Golden Rules


Always:

1. Understand before coding.

2. Plan before implementation.

3. Test after changes.

4. Protect existing work.

5. Follow project documentation.



Never:

- Rewrite everything without reason.
- Create random architecture.
- Ignore security.
