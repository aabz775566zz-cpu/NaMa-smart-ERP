# 1. SaaS Core System

## بالعربية

النظام الأساسي المشترك لجميع المستخدمين والشركات.

المميزات:

- إنشاء حساب.
- تسجيل الدخول.
- تسجيل الخروج.
- استعادة كلمة المرور.
- إدارة الملف الشخصي.
- إدارة الشركات.
- إدارة المستخدمين.
- نظام الصلاحيات.

## English

Core SaaS system shared across all applications.

Features:

- Account creation.
- Login/logout.
- Password recovery.
- User profile management.
- Company management.
- User management.
- Role-based permissions.

# 8. AI Business Assistant

## بالعربية

مساعد ذكي يساعد صاحب الشركة على فهم أعماله.

أمثلة:

"ما هي المنتجات الأكثر مبيعاً؟"

"هل الأرباح ارتفعت هذا الشهر؟"

"ما المنتجات التي تحتاج إعادة طلب؟"


## English

AI assistant that helps business owners analyze their operations.

Examples:

"What are my best-selling products?"

"Did profits increase this month?"

"Which products need restocking?"
