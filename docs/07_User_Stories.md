# ERP Smart User Stories

# قصص المستخدم | User Stories


# 1. Account Management

## تسجيل الحساب


### User Story

As a business owner,
I want to create an account,
so I can start managing my company.


## Acceptance Criteria

- User can register with email.
- User receives confirmation.
- User can login securely.


---

# 2. Company Setup

## إنشاء الشركة


### User Story

As a business owner,
I want to create my company profile,
so the system can be customized for my business.


## Acceptance Criteria

User can add:

- Company name.
- Business type.
- Country.
- Currency.



---

# 3. User Permissions

## إدارة الموظفين


### User Story

As an owner,
I want to control employee permissions,
so each employee can access only allowed features.


## Acceptance Criteria

Owner can:

- Add employees.
- Assign roles.
- Remove access.
